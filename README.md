🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors Game built using Python and Flask with a clean web interface using HTML and CSS.
In this game, the user plays against the computer. The computer randomly selects Rock, Paper, or Scissors, and the winner is determined based on the classic game rules.

This project was developed as part of the CodSoft Python Programming Internship (Task-4).

📌 Features

Interactive web-based game interface

User vs Computer gameplay

Computer generates random moves

Game result display (Win / Lose / Tie)

Score tracking system

Modern UI with HTML & CSS

Multiple rounds gameplay

⚙️ Game Rules

The rules of the Rock Paper Scissors game are simple:

Player Choice	Computer Choice	Result
Rock	Scissors	Player Wins
Rock	Paper	Computer Wins
Paper	Rock	Player Wins
Paper	Scissors	Computer Wins
Scissors	Paper	Player Wins
Scissors	Rock	Computer Wins
Same	Same	Tie

Game logic used in this project follows the traditional rule set of Rock–Paper–Scissors.

▶️ How to Run the Project

Follow these steps to run the project locally.

1️⃣ Install Python

Download Python from:

https://www.python.org/downloads/

Check installation:

python --version
2️⃣ Install Flask

Run the following command:

pip install flask
3️⃣ Clone the Repository
git clone https://github.com/sravya0125/rock-paper-scissors-game.git
4️⃣ Navigate to the Project Folder
cd rock-paper-scissors-game
5️⃣ Run the Application
python app.py
6️⃣ Open in Browser

Open the following link in your browser:

http://127.0.0.1:5000
